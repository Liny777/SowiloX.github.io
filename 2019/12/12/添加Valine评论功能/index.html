添加Valine评论功能

Author: S.X. Date: December 12, 2019  21:45:25 Category: Log

想着给自己博客加个评论功能，奈何主题作者没有给出设置选项，只能自己摸索着配置。

此评论功能采用的是Valine。虽然我更喜欢来必应

注册LeanCloud

首先需要注册一个LeanCloud账号。随便添加一个应用后，可在设置中查看AppID和AppKey。

添加配置

打开themes中的_config.yml文件，添加Valine相关配置：

valine:
    appid:  #你的AppId
    appkey:  #你的AppKey
    verify: true #验证码
    notify: true #评论回复提醒
    placeholder: 欢迎留言 #评论框占位符

创建ejs文件

在/layout/_plugins文件夹下创建名为valine.ejs的文件，并写入下面的代码：

<div class="valine_comment"></div>
<!--Leancloud 操作库:-->
<script src="//cdn1.lncld.net/static/js/3.0.4/av-min.js"></script>

<script src="//unpkg.com/valine/dist/Valine.min.js"></script>
<script>
  new Valine({
      el: '.valine_comment',
      app_id: '<%= theme.valine.appid %>',
      app_key: '<%= theme.valine.appkey %>',
      placeholder: '<%= theme.valine.placeholder %>',
      notify: '<%= theme.valine.notify %>',
      verify: '<%= theme.valine.verify %>',
    });
</script>

添加调用Valine代码

之后需要在相应的生成文章样式的ejs代码中添加调用代码，主题不同位置也会有所不同，我的主题是在layout/_page/post.ejs中添加：

<section>
            <% if (theme.valine && theme.valine.appid && theme.valine.appkey){ %>
            <section id="comments" class="comments">
              <style>
                .comments{margin:30px;padding:10px;background:#fff}
                @media screen and (max-width:800px){.comments{margin:auto;padding:10px;background:#fff}}
              </style>
              <%- partial('_plugins/valine', {
                key: post.slug,
                title: post.title,
                url: config.url+url_for(post.path)
              }) %>
            </section>
        <% } %>
</section>

之后就可以部署进行测试了。
